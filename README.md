## 帷幄（VWeiwo）

AI 沟通教练工具

帷幄（VWeiwo）是一款面向实时沟通场景的 AI 沟通教练工具，适用于面试、商务沟通、咨询答疑等语音或文字对话场景。产品可根据当前问题生成结构化回答建议，帮助用户梳理思路、清晰表达、降低临场紧张感。

## 产品简介

帷幄聚焦沟通辅助场景，强调真实表达、即时参考与低打扰交互体验。用户在对话过程中输入或识别问题后，系统会结合上下文提供简洁版与深度版回答建议，便于快速组织语言并提升表达效率。
