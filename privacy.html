隐私政策

生效日期：2026年6月8日 · 运营者：赤峰松山区帷幄信息技术工作室（个体工商户） · 服务条款

1. 引言

帷幄锦囊（VWeiwo，包名 com.vweiwo.app）由赤峰松山区帷幄信息技术工作室（个体工商户）运营。个人信息相关咨询、查询、更正、删除或注销申请，请通过应用内「设置」与我们联系。

2. 我们收集的信息

  • 账户：邮箱、登录会话；使用支付宝登录/支付时，经您授权获取的必要标识。
  • 语音：麦克风音频，用于转写并生成建议；声纹相关数据默认仅存于本机。
  • 文本与模型：您输入的内容及提示，用于生成建议；可能经您配置的第三方 API 或我们托管通道处理。
  • 订阅：与账户相关的偏好、用量、支付状态等。

应用使用网络与麦克风权限；您可在系统设置中关闭麦克风，部分功能将不可用。

3. 第三方 SDK

主要集成：Capacitor（应用壳）、Supabase 客户端（账户与云同步，数据存于境内阿里云托管）、支付宝 SDK（登录与支付）。可选：端侧大模型、第三方大模型 API。具体名称与字段以各应用商店「第三方共享清单」及您安装的版本为准。

4. 存储与共享

账户与订阅类数据存储于中华人民共和国境内。启用第三方 AI 或语音能力时，相关数据可能按该服务商政策处理（可能在境外），以产品内说明为准。

5. 您的权利

您可依法查询、复制、更正、删除个人信息，或撤回同意。请通过应用内联系我们处理。

6. 其他

未成年人请在监护人同意下使用。我们可能更新本政策，重大变更将通过应用内提示。
